# Genesis Auto — лендинг

Одностраничный сайт на **Vue 3 + Vite + Tailwind CSS v4**.
Премиум-стиль (графит + золото), адаптив, приём заявок в Telegram.

## Запуск

```bash
npm install        # установить зависимости (один раз)
npm run dev        # запустить локально → http://localhost:5173
```

## Сборка для продакшена

```bash
npm run build      # соберёт готовый сайт в папку dist/
npm run preview    # посмотреть собранную версию локально
```

Содержимое папки `dist/` — это и есть готовый сайт.

## Заявка (Яндекс.Форма)

Форма заявки встроена через Яндекс.Формы.
1. forms.yandex.ru → открой свою форму → «Поделиться» → «Установить на сайт».
2. В коде найди ссылку вида `forms.yandex.ru/u/XXXXXX/` и скопируй часть `XXXXXX`.
3. Вставь её в `YANDEX_FORM_ID` в файле `src/components/LeadForm.vue`.

## Объявления из Telegram-канала

Блок «Актуальные объявления» автоматически подтягивает посты (с фото и без)
из вашего публичного Telegram-канала и обновляется каждые 1.5 минуты.

Чтобы это заработало вживую, сайт нужно развернуть на **Vercel** (бесплатно) —
там работает серверная функция `api/telegram.js`, которая и забирает посты:

1. Залей проект на GitHub.
2. На vercel.com → «New Project» → выбери репозиторий (Vercel сам поймёт, что это Vite).
3. В настройках проекта → Environment Variables добавь `TG_CHANNEL` = имя канала без `@`.
4. То же имя канала впиши в `CHANNEL` в `src/components/ListingsSection.vue`.

> Канал должен быть **публичным** (иметь @username). Пока функция не подключена,
> на сайте показывается аккуратный блок с кнопкой «Смотреть объявления в Telegram».
> Локально `npm run dev` функцию не запускает — увидишь именно этот запасной блок.
> Чтобы проверить ленту локально, поставь Vercel CLI и запусти `vercel dev`.
